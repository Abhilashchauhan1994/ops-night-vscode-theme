## ✨ Features

Ops Night is a carefully crafted dark theme inspired by modern developer favorites like Night Owl, Dracula, Atom One Dark, and Tokyo Night — but optimized specifically for infrastructure and production engineering workflows.

### Built for:

- DevOps Engineers
- SREs
- Platform Engineers
- Backend Developers
- Cloud Engineers
- Terminal-first developers

### Optimized for:

- Bash scripting
- Kubernetes YAML
- Terraform
- Dockerfiles
- JSON logs
- Terminal readability
- Long coding sessions
- Low eye strain

# Installation

## From VS Code Marketplace

1. Open Extensions Sidebar
2. Search for: Ops Night
3. Click install
4. Open Command Palette: ctrl + shift + P
5. Preferences: Color Theme
6. Choose : Ops Night
